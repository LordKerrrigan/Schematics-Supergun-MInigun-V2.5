# Schematics-Supergun-MInigun-V2.5
Reverse engineering

Creation of the schematic, based on the project's open-source PCB
https://www.arcade-projects.com/threads/minigun-supergun-an-open-source-supergun.9408/

Note:
* Schematic creation using EasyEDA.
* PDF generation for routing the various components.
* The voltmeter is not included in the schematic.
* The footprint of the A6H-4101 is not the same between the PCB and SCH models.
